## RECORDING TIPS

### Setup:
1. Clean terminal with dark theme
2. Large font (18-20pt) for readability
3. Hide desktop clutter
4. Test audio levels

### Recording Tools:
- **Screen**: OBS Studio (free) or QuickTime (Mac)
- **Audio**: Built-in mic is fine, but use headset if possible
- **Editing**: DaVinci Resolve (free) or iMovie (Mac)

### B-Roll Ideas:
- Architecture diagrams
- Code snippets (briefly)
- Test results
- Web UI interactions
- Terminal animations

### Music:
- Use royalty-free music from YouTube Audio Library
- Keep volume low (background only)
- Upbeat but professional

### Pacing:
- Speak clearly and not too fast
- Pause between sections
- Show, don't just tell
- Keep energy high

## QUICK RECORDING SCRIPT

If short on time, record just the terminal demo:

```bash
# Terminal recording with asciinema
brew install asciinema
asciinema rec demo.cast

# Run demo
python demo_hackathon.py

# Stop recording (Ctrl+D)

# Convert to GIF
npm install -g asciicast2gif
asciicast2gif demo.cast demo.gif
```

Then add text overlays in any video editor.
